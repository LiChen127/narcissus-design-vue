|  ----  | ----  | ----  | ----  | ----  |
|  type| 定义按钮类型  | String  | primary, dashed, warn, error|  |
|  disabled| 定义按钮是否禁用  | Boolean  | true, false | false |
|  size| 定义按钮的大小  | String  | mini, small, medium |  |
|  loading| 定义加载状态  | Boolran  | true, false | false |
